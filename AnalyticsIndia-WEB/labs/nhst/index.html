Hypothesis Testing

Signal vs. Noise

Science is about distinguishing a real effect (Signal) from random chance (Noise). But there is always a trade-off.

The Balancing Act

If you lower your standard of evidence (Alpha), you catch more discoveries, but you also make more mistakes (False Positives).

Type I Error (Alpha)

"Crying Wolf." You say there is an effect when there isn't one. Usually set to 5% (0.05).

Type II Error (Beta)

"Missing the Ghost." There WAS a real effect, but your experiment wasn't sensitive enough to find it.
